<h1>Установка и использование</h1>

  Во всех вариантах установки требуется Node.js с npm.
  По умолчанию сервис запускается на localhost:3000.
  Для корректной работы на localhost:8080 нужно запустить бэкенд River совместимой версии.

<h3>Со страницы проекта (при отсутствии подходящего релиза)</h3>
   <ol>
  <li>Клонировать или скачать проект</li>
  <li>Запустить install_self_and_pm2.cmd (pm2 нужен для запуска сервиса)</li>
     <li>Запустить build.cmd</li>
     <li>Запуск:</li>
      <ul>
         <li>Запустить run.cmd</li>
        <li>Для приостановки процесса запустить stop.cmd</li>
   <li>Для удаления процесса запустить kill.cmd.</li></ul>
    </ol>
  
  

<h3>Для использования (1.0.0+):</h3>
<ol>
  <li>Скачать Riverfront.zip со страницы нужного релиза и распаковать архив.</li>
  <ul>
  <li>Для запуска в консоли запустить run.cmd, входящий в состав Riverfront</li>
   <li>Для запуска процесса убедиться, что на устройстве установлен менеджер процессов pm2 (запустить install_pm2 при необходимости), и запустить shadow_run</li>
   <li>Для приостановки процесса запустить stop.cmd</li>
   <li>Для удаления процесса запустить kill.cmd.</li></ul>

  </ol>
