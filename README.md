<h1>Установка и использование</h1>

  Во всех вариантах установки требуется Node.js с npm.

<h3>Для разработки</h3>
   <ol>
  <li>Загрузить и разархивировать проект</li>
  <li>Из директории проекта запустить npm install (эта команда установит зависимости)</li>
  <li>Запустить npm run build</li>
  <li>запустить npm install -g pm2</li>
  <li>запустить  pm2 start ecosystem.config.js</li>
  <li>Фронтенд запущен на localhost:3000.</li>
     </ol>
  Для корректной работы нужно запустить на localhost:8080 бэкенд River совместимой версии.
  

<h3>Для использования (1.0.0+):</h3>
<ol>
  <li>Скачать Riverfront.zip со страницы релиза и распаковать архив.</li>
  <ul>
  <li>Для запуска в консоли запустить run.cmd, входящий в состав Riverfront</li>
   <li>Для запуска процесса убедиться, что на устройстве установлен менеджер процессов pm2 (запустить install_pm2 при необходимости), и запустить shadow_run</li>
   <li>Для приостановки процесса запустить stop.cmd</li>
   <li>Для удаления процесса запустить kill.cmd.</li></ul>

  </ol>
